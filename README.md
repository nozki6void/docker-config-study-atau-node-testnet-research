# Docker Configuration Study 🐳
Catatan ini berisi perintah Docker yang sering digunakan untuk menjalankan node airdrop di VPS.

### Perintah Docker Utama:
- `docker-compose up -d`: Menjalankan node di latar belakang.
- `docker ps`: Melihat kontainer node yang sedang aktif.
- `docker logs -f [container_id]`: Memantau log jalannya node secara real-time.
- `docker-compose down`: Menghentikan layanan node.

### Dokumentasi Pribadi:
- Mempelajari cara instalasi Docker Engine di Ubuntu.
- Memahami penggunaan port mapping untuk node network.
